# Project Title: Website

## Overview
This project is a static website designed to serve as a landing page for various stakeholders including users, landlords, councils, and investors. The website is set to launch in Q3 2026 and aims to comply with the guidelines of both the Apple App Store and Google Play Store.

## Project Structure
```
Website
├── index.html        # The landing page of the website
├── css
│   └── styles.css    # Styles for the website
├── js
│   └── script.js     # JavaScript functionality
├── assets
│   └── fonts         # Font files for consistent typography
└── README.md         # Documentation for the project
```

## Features
- **User Sections**: Information tailored for users, landlords, councils, and investors.
- **Responsive Design**: The website is designed to be responsive and user-friendly across all devices.
- **Compliance**: Adheres to the requirements of the Apple App Store and Google Play Store.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the landing page.
3. Modify the `css/styles.css` file to customize styles as needed.
4. Use the `js/script.js` file to add any interactive functionality.

## Launch Date
The website is scheduled to launch in Q3 2026. Stay tuned for updates!
